# 📌 Project Overview

This repository contains a **complete learning series of Matplotlib**, the most popular Python library for **data visualization**.

Matplotlib allows users to create:

- Line graphs
- Scatter plots
- Bar charts
- Histograms
- Pie charts
- Subplots and multiple charts

The tutorials demonstrate how to build **clear and informative visualizations for data analysis and data science projects**.

---

# 🧠 How It Works

Each Python file demonstrates **a specific Matplotlib visualization concept**.

Learning Flow:

1. Matplotlib introduction
2. Basic plotting with Pyplot
3. Line charts and markers
4. Labels and titles
5. Gridlines and styling
6. Subplots
7. Scatter plots
8. Bar charts
9. Histograms
10. Pie charts

These tutorials help learners understand **how to visualize datasets effectively using Python**.

# ⚙ Installation

## 🪟 Windows

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/matplotlib-python-tutorials.git
```

### 2. Go to the project folder

```bash
cd matplotlib-python-tutorials
```

### 3. Install required packages

```bash
pip install matplotlib numpy
```

### 4. Run the example file

```bash
python 1_intro.py
```

---

## 🐧 Linux

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/matplotlib-python-tutorials.git
```

### 2. Go to the project folder

```bash
cd matplotlib-python-tutorials
```

### 3. Install dependencies

```bash
pip3 install matplotlib numpy
```

### 4. Run the example file

```bash
python3 1_intro.py
```
